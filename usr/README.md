# Trabajo Práctico: Producción de la Patagonia Extrandina

Esta aplicación es un reporte formal y académico sobre la economía y producción de la región de la Patagonia Extrandina en Argentina, desarrollado para presentar información estadística clave, comparativas temporales y gráficos interactivos.

## Resumen del Proyecto

La aplicación funciona como un trabajo práctico interactivo que analiza los principales pilares productivos de la meseta patagónica (hidrocarburos, ganadería ovina, energías renovables) y compara su evolución en un período de 10 años (2013 vs 2023).

### Características Principales

*   **Introducción Formal:** Contexto geográfico y económico de la Patagonia Extrandina.
*   **Datos Estadísticos:** Puntos clave sobre la producción petrolera, el stock ovino y el desarrollo eólico, con fuentes citadas.
*   **Cuadro Comparativo:** Tabla de datos interactiva que muestra la variación a 10 años de los principales indicadores.
*   **Gráfico de Barras Interactivo:** Visualización comparativa de la producción de hidrocarburos utilizando la biblioteca `fl_chart`.
*   **Citas y Fuentes:** Todo el contenido incluye la fuente oficial correspondiente al final de cada sección (INDEC, Secretaría de Energía, SENASA).

## Tecnologías Utilizadas

*   **Flutter:** Framework principal para la interfaz multiplataforma.
*   **Dart:** Lenguaje de programación.
*   **fl_chart:** Para la renderización de gráficos de barras estadísticos.

## Instrucciones de Ejecución

1.  Asegúrate de tener instalado el SDK de Flutter.
2.  Clona este repositorio o descarga el código fuente.
3.  Abre una terminal en la raíz del proyecto y ejecuta el siguiente comando para obtener las dependencias:
    ```bash
    flutter pub get
    ```
4.  Ejecuta la aplicación en tu dispositivo o emulador preferido:
    ```bash
    flutter run
    ```
